AI-Machine Learning

Machine learning is a branch of artificial intelligence which simulates that systems can learn from information, data, learn patterns, identify patterns and make decisions without the need of human intervention. This is a method of data analysis which is autonomously used in model building, surveillance, intelligence gathering and is a part of a huge growing market in the evolution of technology assisted devices. This is a dynamic tool which can modify itself when being exposed to new data, and does not require human intervention to adapt to its environment.

Tom Mitchell - “A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P if its performance at tasks in T, as measured by P, improves with experience E.”
